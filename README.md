# java-shooting

【概要】
本プロジェクトは、JavaのSwingを用いた基礎的なシューティングゲームです。大学1年時に講義でJavaの基礎を学んだ際、その応用として作成いたしました。Java Swingを用いた初歩的なGUI構築ができることを示すための制作物です。

【工夫したポイント】
・自機のマウスポインタ追従機能の実装
・敵機およびその弾の動きにおける物理演算の導入
単なる数値の書き換えではなく、数学的な基礎に基づいた物体の動きをプログラムに落とし込むことに注力しました。シンプルな構成ながら、操作性と動的な計算処理の連動を意識して制作しています。

【改善点】
現在は仮想ウィンドウ（GUI環境）を構築しない限りGitHub上で直接実行できないという制約があります。今後は、ブラウザ上で動作するライブラリへの移行や、より高度な物理ライブラリを使わずに自力で計算式を拡張し、複雑な敵のアルゴリズムを実装することを目指しています。

【実行方法】
GitHub上ではそのまま実行できないため、ローカル環境での実行が必要です。
1 本リポジトリをダウンロード（またはクローン）
2 デスクトップなど任意の場所に配置
3 フォルダをエクスプローラーで開く
4 コマンドプロンプトを起動
5コンパイルおよび実行
